## 关于般若
> “般若”，《六祖坛经》解说：“何名般若？般若者，唐言智慧。一切处所，一切时中，念念不愚，常行智慧，即是般若行。一念愚即般若绝;一念智即般若生。世人愚迷，不见般若。口说般若，心中常愚，常自言我修般若，念念说空，不识真空。般若无形相，智慧心即是。若作如是解，即名般若智。
